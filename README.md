# Dropbox Colors

This demo is made with [Flutter][flutter].

This project was inspired by the [Color page][dropbox-color] from the [Dropbox Brand Guidelines][dropbox-brand] website.

[Live demo][live-demo] is deployed to GitHub Pages with the help of the [peanut][peanut] package.

[dropbox-color]: https://brand.dropbox.com/color

[peanut]: https://pub.dev/packages/peanut
